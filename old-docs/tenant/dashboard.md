Dashboard utama tenant dengan overview kontrak, tagihan, dan aktivitas.

## File Location
- `src/pages/tenant/Dashboard.tsx` - Halaman dashboard
- `src/components/layouts/TenantLayout.tsx` - Layout

## Database Tables
- `contracts` - Kontrak aktif
- `invoices` - Tagihan

# Merchant Referrals

## Overview
Program referral untuk merchant.

## File Location
- `src/pages/merchant/Referrals.tsx` - Halaman referrals
- `src/components/referral/ReferralDashboard.tsx` - Dashboard

## Database Tables
- `referral_codes` - Kode referral
- `referral_rewards` - Reward
- `referrals` - Relasi referrer-referee
- `merchant_referral_summary` - View ringkasan referral per merchant

# Tenant Forum

## Overview
Forum diskusi antar tenant dalam properti yang sama.

## File Location
- `src/pages/tenant/Forum.tsx` - List posts
- `src/pages/tenant/ForumPost.tsx` - Detail post

## Database Tables
- `forum_posts` - Posts
- `forum_comments` - Comments
- `forum_likes` - Likes
- `forum_reports` - Reports
